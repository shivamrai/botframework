    ROCm runtime & dev (follow AMD repo for Fedora — example placeholder steps)

    Add AMD/ROCm repo per AMD instructions for your distro (replace with the current AMD repo URL).
    Then install ROCm packages:

# after adding AMD repo per AMD docs
dnf install -y rocm-dev hip-rocclr hip-runtime rocm-opencl rocm-libs rocblas rocm-dev
# enable ROCm paths
echo 'export PATH=/opt/rocm/bin:$PATH' >> /etc/profile.d/rocm.sh
source /etc/profile.d/rocm.sh

    Verify ROCm / HIP visible

/opt/rocm/bin/rocminfo
/opt/rocm/bin/rocm-smi
hipcc --version

    Create Python venv and install lightweight test deps

source ~/venvs/bot-dev/bin/activate
pip install --upgrade pip setuptools wheel
# example small inference libs for quick tests
pip install numpy torch --index-url https://download.pytorch.org/whl/rocm5.2  # adjust for your ROCm version

    Quick clone & build example native backend (llama.cpp-style)

git clone https://github.com/ggerganov/llama.cpp.git
cd llama.cpp
# clean previous CMake build artifacts (safe if build dir exists)
cmake --build build --target clean || true
# configure HIP/ROCm build
cmake -S . -B build -DGGML_HIP=ON -DCMAKE_BUILD_TYPE=Release
# build all targets
cmake --build build -j$(nproc)
# run a small test command (prints help)
./build/bin/llama-cli -h
